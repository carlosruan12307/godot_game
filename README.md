# Introdução ao nosso jogo

## Controles Básicos

- **Setas**: Movimentação do personagem
- **Espaço**: Pular
- **Z**: Atacar
- **C**: Deslizar

## Controles Especiais

### Ao Pegar o Arco

- **Q**: Atirar flechas
- **E / W / A / S**: Mover o arco para ajustar o ângulo de disparo

Divirta-se jogando!
